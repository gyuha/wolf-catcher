# Wolf catcher

## Requirements

- pyenv 
  - local version : 3.8.10
  - [GitHub - pyenv/pyenv: Simple Python version management](https://github.com/pyenv/pyenv)
- poetry
  - https://python-poetry.org/docs/

## Before start

```bash
peotry install
```
